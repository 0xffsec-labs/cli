# Remote Codex CLI Releases

This repository publishes macOS and Linux binaries for Remote Codex. The source
code and release automation are maintained in a separate private repository.

After the first stable release is published, install it with:

```sh
curl -fsSL https://raw.githubusercontent.com/0xffsec-labs/cli/main/install.sh | bash
```

Current releases support Apple Silicon macOS and x86_64 Linux. Each release also
includes a SHA-256 checksum manifest. Remote Codex is based on OpenAI Codex and
is distributed under the Apache License 2.0; see `LICENSE` and `NOTICE`.
